# my-visual-tools

小学数学可视化教学工具库。

本仓库用于整理课堂教学中使用的 HTML 可视化工具，主要服务于小学数学教学、项目化学习、课堂演示、学生交互探究与练习讲评。当前内容以五年级下册数学为主，重点覆盖长方体与正方体、体积、排水法、分数、公因数、公倍数等内容。

---

## 项目定位

这个仓库主要用于沉淀以下类型的教学工具：

- 课堂投屏演示工具
- 学生交互式学习单
- 数学概念动态可视化页面
- 教材与作业本配套练习页面
- 项目化学习任务情境页面
- 公开课、区级汇报、教研展示辅助材料

这些工具适合在数学课堂中用于：

1. 将抽象数学概念具体化、动态化；
2. 支持学生观察、猜想、验证和表达；
3. 辅助教师完成重点环节的可视化讲解；
4. 提升课堂互动性和教学展示效果。

## 使用方式

### 方式一：直接打开 HTML 文件

1. 下载仓库中的 `.html` 文件；
2. 使用浏览器打开，例如 Chrome、Edge、Safari；
3. 连接投屏设备后即可在课堂中使用。

### 方式二：课堂投屏使用

建议使用方式：

- 教师电脑提前打开对应 HTML 页面；
- 全屏显示浏览器；
- 根据课堂节奏点击、拖动或逐步展示；
- 配合板书、任务单或学生操作材料使用。

### 方式三：作为公开课或汇报素材

这些页面也可以作为：

- 公开课技术支持材料；
- AI 辅助教学案例素材；
- 教研汇报中的课堂实践证据；
- 数学可视化教学资源库的一部分。
