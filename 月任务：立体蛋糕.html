🎂 立体蛋糕解析器

示例数据：底层(20×20×8), 中层(14×14×8), 顶层(8×8×8)

交互探索

无论如何堆叠，总体积等于三个长方体体积之和。

V = V₁ + V₂ + V₃
V = (20×20×8) + (14×14×8) + (8×8×8)
V = 3200 + 1568 + 512 = 5280 cm³

解法一：扣除法（点击"拆解模型"观察）

深灰色代表被遮挡不涂奶油的面：底面、两次重叠面。

S = 三表面积总和 - (底面 + 2×下重叠 + 2×上重叠)
重叠扣除 = 400 + 2×(14×14) + 2×(8×8) = 920
S = (1440+840+384) - 920 = 1744 cm²

解法二：投影法（点击"俯视投影"观察）

从上方看，三个露出的顶面拼起来刚好等于最底层顶面。因此涂色面积 = 最底面顶面积 + 三个盒子的侧面积之和。

S = 底层顶面 + (S侧₁ + S侧₂ + S侧₃)
S = (20×20) + (640 + 448 + 256)
S = 400 + 1344 = 1744 cm²
